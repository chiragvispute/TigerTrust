# Required Image Assets

Please add the following SVG files to this directory:

1. `crypto-wallet-illustration.png` - Used on the landing page hero section
2. `success-checkmark.png` - Used on the identity verification success page

These files are required for the application to display properly. You can create your own PNG files or find suitable illustrations online.

**File Requirements:**
- Format: PNG
- crypto-wallet-illustration.png: Recommended size ~500x400px
- success-checkmark.png: Recommended size ~80x80px

The application will not display images correctly until these files are added.
